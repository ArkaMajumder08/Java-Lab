Program 1 – Rectangle Class (Area & Perimeter)
Write a program for a Rectangle class that takes length and width of a rectangle as input and calculates area and perimeter.

Code:
class rectangle
{
void area(int a, int b)
{
System.out.println("Area = " + (a * b));
}

void perimeter(int a, int b)
{
System.out.println("Perimeter = " + (2 * (a + b)));
}

public static void main(String str[])
{
rectangle ob = new rectangle();
int l = Integer.parseInt(str[0]);
int w = Integer.parseInt(str[1]);
ob.area(l, w);
ob.perimeter(l, w);
}
}

Output:
$ java rectangle 2 3
Area = 6
Perimeter = 10
