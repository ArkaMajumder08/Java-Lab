Program 4 – Constructor Overloading: Cone Volume
Write a class Cone that calculates the volume of a cone. Overload the constructor by passing height and radius as arguments, and also with no arguments.

Code:
class Cone
{
double r, h;
double pi = 3.14;

Cone()   // no argument constructor
{
r = 1;
h = 1;
}

Cone(double radius, double height)   // parameterized constructor
{
r = radius;
h = height;
}

void volume()
{
double v = (pi * r * r * h) / 3;
System.out.println("Volume of Cone = " + v);
}

public static void main(String[] args)
{
Cone c1 = new Cone();
c1.volume();
double r = Double.parseDouble(args[0]);
double h = Double.parseDouble(args[1]);
Cone c2 = new Cone(r, h);
c2.volume();
}
}

Output:
$ java Cone 3.0 5.0
Volume of Cone = 1.0466666666666667
Volume of Cone = 47.1
